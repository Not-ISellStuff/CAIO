# CAIO
Tool for account cracking

Modules:

1. Multiple Checkers
2. Combolist Tools
3. Hits Manager
4. Settings

# How To Use

First of all make sure you have Python 3.12.

1. Open Up setup.bat | This will install some libraries
2. Put your combolist in combo.txt
3. Put your proxies in proxies.txt
4. open caio.py
